# akka services data get-task

Get a data operation task for a service

## Synopsis

The `akka service data get-task` command gets a data task for a service.

```
akka services data get-task SERVICE TASK_UUID [flags]
```

## Options

```
  -h, --help             help for get-task
      --owner string     the owner of the project to use, needed if you have two projects with the same name from different owners
      --project string   project to use if not using the default configured project
      --region string    region to use if project has more than one region
```
